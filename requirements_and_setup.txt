## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Place your trained `model.pt` in the project directory
3. Run the application: `python space_station_app.py`
4. Load test images or start camera feed for detection

## Features
- Real-time object detection for space station equipment
- Automated monitoring and logging
- Falcon platform integration for model updates
- Professional space-themed interface
- Export capabilities for data analysis

## System Requirements
- Python 3.8+
- 8GB RAM (16GB recommended)
- GPU with 4GB VRAM (optional)
- Windows 10/11, macOS 10.14+, or Linux Ubuntu 18.04+

## Model Files
Ensure your trained model (`model.pt`) is in the root directory for automatic loading.

## Documentation
See `Space_Station_Detection_Proposal.pdf` for complete technical documentation and Falcon integration details.
